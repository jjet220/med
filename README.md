# Medical

Medical - это бесплатная программа регистрации и управления пациентами.

## О проекте

Приложение работает как на Linux так и на Windows.

### Возможности

 - **Управление пациентами** - создание, редактирование, поиск, удаление пациентов
 - **Валидация данных** - проверка телефонов, email, СНИЛС и номер полиса ОМС
 - **REST API** - большое количество эндпоинтов для интеграции с другими системами
 - **Пагинация** - страничная навигация по пациентам
 - **Фильтрация** - филтрация по ФИО, дате рождения, полу, телефону, email, СНИЛС, полису
 - **Центральная обработка ошибок** - единое место в коде с ответом об ошибках

## Технологии

 - **Java 26** - Основной язык программирования
 - **Spring Boot 4.0.5** - Фрейморк для построения веб-приложений
 - **Spring Web MVC** - REST API контроллеры
 - **Spring Data JPA** - Работа с базой данных
 - **Spring Validation** - Валидация входящих данных
 - **PostgreSQL 18** - Реляционная база данных
 - **Liquibase** - Управление миграциями БД
 - **Lombok** - Уменьшение шаблонного кода
 - **Maven** - Сборка проекта

## Системные требования

 - **Java:** JDK 26 или новее
 - **PostgreSQL:** 18 или новее
 - **ОС:** Windows / Linux 
 - **Память:** минимум 512MB RAM
 - **Диск:** минимум 500MB свободного места

## Коды ошибок

### Ошибки валидации (400 Bad Request)

- `VALIDATION_SURNAME`  Фамилия обязательна (1-80 символов) 
 - `VALIDATION_NAME`  Имя обязательно (1-80 символов) 
 - `VALIDATION_DATE_OF_BIRTH`  Дата рождения обязательна 
 - `VALIDATION_SEX`  Пол обязателен 
 - `VALIDATION_PHONE_NUMBER`  Неверный формат телефона (10-15 цифр, может начинаться с +) 
 - `VALIDATION_EMAIL`  Неверный формат email 
 - `VALIDATION_SNILS`  Неверный формат СНИЛС (11 цифр) 
 - `VALIDATION_SINGLE_POLICY_NUMBER`  Неверный формат полиса (16 цифр) 

### Ошибки параметров запроса (400 Bad Request)

 - `INVALID_PARAMETER`  Неверный тип параметра запроса 
 - `MISSING_PARAMETER`  Отсутствует обязательный параметр 
 - `INVALID_JSON`  Неверный формат JSON 
 - `INVALID_DATE_FORMAT`  Неверный формат даты (используйте гггг-ММ-дд) 

### Ошибки ресурсов (404 Not Found)

 - `RESOURCE_NOT_FOUND`  Ресурс не найден 

### Ошибки конфликта (409 Conflict)

 - `DUPLICATE_EMAIL`  Email уже существует 
 - `DUPLICATE_PHONE`  Телефон уже существует 
 - `DUPLICATE_SNILS`  СНИЛС уже существует 
 - `DUPLICATE_POLICY_NUMBER`  Номер полиса уже существует 

### Ошибки сервера (500 Internal Server Error)

 - `INTERNAL_SERVER_ERROR`  Внутренняя ошибка сервера
